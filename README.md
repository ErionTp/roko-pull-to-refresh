# roko-pull-to-refresh

A custom pull to refresh component

## Installation

```sh
npm install roko-pull-to-refresh
```

## Usage

```js
import { multiply } from 'roko-pull-to-refresh';

// ...

const result = await multiply(3, 7);
```
